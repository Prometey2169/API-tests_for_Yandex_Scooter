# Api тесты для сервиса Яндекс Самокат
# Задачи проекта
**1. Протестировать эндпоинты для:** 
- Создания курьера
- Авторизации курьера
- Удаления курьера
- Создания заказа
- Принятия заказа
- Списка заказов
- Получения заказа по его номеру
2. Построение отчёта при помощи Allure
# Инструменты
- Java 11
- Maven
- JUnit 4
- REST Assured
- Allure
- Gson
